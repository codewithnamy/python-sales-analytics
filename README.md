# Cosmetics Sales & Customer Analytics Using Python

## Project Overview
This project performs Exploratory Data Analysis (EDA) on a cosmetics sales dataset to uncover business insights related to sales performance, customer behavior, product categories, pricing trends, and customer ratings. The analysis demonstrates data cleaning, visualization, and analytical techniques using Python.

## Objectives
- Analyze cosmetics sales data to identify key sales trends.
- Understand customer purchasing behavior.
- Evaluate product category performance.
- Explore the relationship between product pricing and customer ratings.
- Generate actionable business insights through data visualization.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Analysis Performed
- Data cleaning and preprocessing
- Handling missing values and inconsistent data
- Exploratory Data Analysis (EDA)
- Sales trend analysis
- Product category analysis
- Customer rating analysis
- Price distribution analysis
- Data visualization using charts and graphs

## Key Insights
- Identified top-performing product categories based on sales.
- Analyzed customer rating patterns across products.
- Explored pricing trends and their relationship with customer ratings.
- Generated visual reports to support data-driven decision-making.

4. Run all cells to reproduce the analysis.
